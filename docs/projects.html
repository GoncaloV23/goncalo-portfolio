Knapsack Parallel Solver

- Repository:

  Click Here

- Description:

  This project involves the implementation of the AJ-KP-BS algorithm to solve the 0-1 knapsack problem, using parallel and concurrent programming techniques in Java. The goal is to maximize the total value of selected items while ensuring that the total weight does not exceed the knapsack's capacity. The solution includes the creation of multiple threads executing the AJ-KP-BS algorithm, sharing information about the best solutions found. Proper synchronization between threads is essential to avoid conflicts when accessing shared memory. The results are presented in tables that include the test number, test name, execution time, number of threads used, best value found, total weight of the best solution, number of iterations, and time to reach the best solution. The project is developed in Java, utilizing parallel and concurrent programming techniques such as threads and semaphores.

- Technologies:

  IntelliJ (Java)
